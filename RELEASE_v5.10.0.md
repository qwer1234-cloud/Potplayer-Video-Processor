# Video Processing Tool Beta v5.10.0 Release

## 📋 Release Overview

**Release Date**: 2025-01-08
**Version**: 5.10.0
**Type**: Feature Enhancement
**Status**: ✅ Ready for Release

## 🎯 主要新功能

### 🗂️ 批量文件前缀处理功能
- **批量添加前缀**：为选定文件夹下的所有文件添加指定前缀
- **批量删除前缀**：移除选定文件夹下所有文件的指定前缀
- **智能文件处理**：只处理文件（不包含文件夹），没有匹配到就不处理
- **容错机制**：单个文件处理失败不影响其他文件

### 📊 用户界面改进
- **界面标签更新**：
  - "Output Format" → "Function List"
  - "Video File" → "File List/Folder"
- **默认路径调整**：文件选择默认路径改为E盘
- **美观的输入弹窗**：现代化的弹窗设计，支持多种操作方式

## 🎨 用户体验升级

  - 点击弹窗外部区域取消

- **控制台日志**：详细的操作记录便于调试

## 🚀 使用指南

### 批量前缀处理步骤

1. **启动应用程序**
2. **选择功能**：
   - 在"Function List"中选择"批量添加前缀"
   - 或选择"批量删除前缀"
3. **选择文件夹**：
   - 点击"Browse..."按钮
   - 默认打开E盘
   - 选择包含待处理文件的文件夹
4. **输入前缀**：
   - 点击"Start Processing"
   - 在弹窗中输入要处理的前缀
   - 按Enter确认或Esc取消
5. **等待处理完成**：
   - 实时查看处理进度
   - 查看最终结果统计

### 功能特性

#### 智能匹配和处理
- **添加前缀**：
  - 跳过已有此前缀的文件
  - 新增前缀到所有匹配文件
- **删除前缀**：
  - 只删除完全匹配的前缀
  - 跳过没有此前缀的文件
- **安全处理**：
  - 不处理子目录
  - 避免文件名冲突
  - 保持文件完整性

#### 错误处理机制
- **文件访问错误**：记录并跳过无法访问的文件
- **权限问题**：处理文件权限不足的情况
- **磁盘空间**：检查重命名操作的空间需求
- **并发控制**：避免同时修改同一文件
