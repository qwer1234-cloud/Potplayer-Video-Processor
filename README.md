# 视频处理工具

一个基于Electron的视频片段提取、GIF制作和文件压缩工具。

## 功能特性

- 🎥 支持多种视频格式 (MP4, AVI, MOV, MKV, WMV, FLV, WebM)
- 🖼️ 支持输出GIF和视频片段
- 📁 **7Zip文件夹压缩功能** (v5.7.0新增)
- 🔖 基于PotPlayer书签(.pbf)的批量GIF生成
- 📝 字幕提取功能
- ⏱️ 精确的时间控制 (小时:分钟:秒:毫秒)
- ⏰ 可调节截取时长 (1-100分钟)
- 🎨 现代化用户界面

## 系统要求

- Windows 操作系统
- Node.js 16.0 或更高版本
- FFmpeg (用于视频处理)
- gifski (用于GIF生成)
- **7Zip命令行工具** (用于7Zip压缩功能)

## 安装步骤

1. 安装Node.js: https://nodejs.org/
2. 安装FFmpeg并添加到系统PATH
3. 安装gifski
4. **安装7Zip并添加到系统PATH** (v5.7.0新增要求)
5. 在项目目录中运行以下命令：

```bash
cd D:\ProcessVideo-Beta
npm install
```

## 使用说明

### 视频处理功能
1. **选择视频文件**: 点击"浏览..."按钮选择视频文件
2. **选择输出格式**: 选择GIF、Video、根据书签截取GIF或提取字幕
3. **设置开始时间**: 使用下拉框选择小时、分钟、秒，输入框输入毫秒
4. **设置截取时长**: 选择1-100分钟的时长
5. **开始处理**: 点击"开始处理"按钮

### 7Zip压缩功能 (v5.7.0新增)
1. **选择输出格式**: 在下拉菜单中选择"7Zip压缩文件夹"
2. **选择文件夹**: 点击"浏览..."按钮选择要压缩的文件夹（默认打开D盘）
3. **开始压缩**: 点击"开始处理"按钮
4. **等待完成**: 压缩进度会实时显示，完成后压缩包保存在D盘根目录

### 书签GIF生成功能
1. **选择书签文件**: 选择"根据书签截取GIF"格式
2. **选择PBF文件**: 支持选择多个PotPlayer书签文件(.pbf)
3. **自动生成**: 系统会根据书签对自动生成多个GIF文件

## 输出目录

- GIF文件输出到: `D://picture12` 或根据PBF文件名的自定义目录
- 视频片段输出到: `D://video_clips`
- **7Zip压缩包输出到**: `D://{文件夹名}.7z`
- **分卷文件自动整理到**: `D:\picture\{分卷号}\` (如001→\001\，002→\002\)
- 字幕文件输出到: 视频文件同目录

### GIF文件命名规则 (v5.6.0更新)
- 格式: `视频名_开始时间.gif`
- 时间格式: `小时-分钟-秒` (去除前导零和毫秒)
- 示例: `myVideo_1-30-15.gif` (表示开始时间01:30:15)

### 7Zip压缩特性
- **压缩级别**: 最高 (-mx9)
- **分卷大小**: 100MB
- **分卷格式**: `{文件夹名}.7z.001`, `{文件夹名}.7z.002`, ...
- **格式**: 7z格式
