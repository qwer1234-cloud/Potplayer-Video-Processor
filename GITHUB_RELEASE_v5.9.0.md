## 🎉 Video Processing Tool Beta v5.9.0

### ✨ 主要新功能

**🗂️ 智能文件整理功能**
- 分卷压缩完成后自动移动所有分卷文件到D:\picture目录
- 根据分卷后缀智能分类：001→\001\，002→\002\，依此类推
- 自动创建子目录：如果目标子目录不存在则自动创建
- 完美兼容现有picture目录结构（001-018文件夹）

**📁 完整处理流程**
1. GIF生成 → 7Zip压缩 → 自动移动 → 文件夹清理
2. 实时移动进度显示：展示文件移动到各子目录的进度
3. 详细结果展示：显示每个分卷的移动详情和文件大小

**🛡️ 安全可靠**
- 完善的错误处理：单个文件移动失败不影响其他文件
- 容错设计：移动失败不影响压缩和清理流程
- 详细日志记录：便于问题诊断和调试

### 🔧 使用方法
1. 选择"根据书签截取GIF"功能
2. 选择PBF书签文件
3. 点击"开始处理"
4. 等待自动完成：GIF生成 → 压缩 → 移动 → 清理
5. 查看整理好的分卷文件：`D:\picture\{分卷号}\`

### 📦 文件输出
**新的文件组织结构**：
```
D:\picture\
├── 001\  ← video.7z.001
├── 002\  ← video.7z.002
├── 003\  ← video.7z.003
└── ...
```

### 🚀 v5.9.0 核心改进
- **自动整理**：无需手动移动，分卷文件自动归档
- **智能分类**：根据分卷号自动分配到对应子目录
- **无缝集成**：与现有压缩和清理流程完美结合
- **详细反馈**：实时显示移动进度和最终结果
