# Video Processing Tool Beta - 发布说明

## 版本 1.0.0-beta (2025-10-26)

### 🎯 主要功能

1. **视频片段提取** - 从指定时间点开始，提取指定时长的视频片段
2. **GIF生成** - 将视频片段转换为高质量GIF动画
3. **批量GIF生成** - 基于PotPlayer书签文件(.pbf)批量生成多个GIF
4. **字幕提取** - 从视频文件中提取字幕流并保存为SRT格式

### ✨ 本次更新内容

#### 🔧 修复的问题
- **修复了GIF文件输出路径问题** - 现在所有GIF文件都会正确保存到D盘根目录
- **统一了GIF生成系统** - 所有GIF生成都使用相同的FFmpeg处理引擎，确保输出一致性
- **改进了批量处理逻辑** - 基于书签的批量GIF生成现在更加稳定可靠

#### 🚀 改进的功能
- **更准确的输出路径控制** - 所有输出文件现在都会保存到指定的D盘根目录
- **增强了错误处理** - 提供更详细的错误信息和处理状态反馈
- **优化了内存管理** - 大批量处理时内存使用更加高效
- **改进了进度显示** - 实时显示处理进度和状态信息

### 📁 文件说明

#### 构建输出
- `Video Processing Tool Beta 1.0.0-beta.exe` - 便携版应用程序（推荐）
- `Video Processing Tool Beta-win32-x64/` - 完整的应用程序文件夹

#### 使用方法
1. **单次GIF生成**:
   - 选择视频文件
   - 选择"GIF"格式
   - 设置开始时间和持续时间
   - 点击"开始处理"

2. **批量GIF生成**:
   - 选择"根据书签截取GIF(支持多文件)"格式
   - 选择一个或多个.pbf书签文件
   - 点击"开始处理"

3. **视频片段提取**:
   - 选择视频文件
   - 选择"Video"格式
   - 设置开始时间和持续时间
   - 点击"开始处理"

### 💡 输出位置
- 所有GIF文件: `D:\`
- 所有视频片段: `D:\`
- 字幕文件: `D:\`

### ⚙️ 系统要求
- Windows 10/11 (64位)
- FFmpeg (需要已安装并在PATH中)
- 足够的磁盘空间用于输出文件

### 🐛 已知问题
- 首次启动时可能会有缓存相关警告（不影响功能）
- 处理超大视频文件时可能需要较长时间

### 📞 支持
如有问题，请检查D盘根目录是否有相应的输出文件生成。
